Чтобы использовать автоматическую авторизацию нужно:
1)Зайти на сайт или в программу, где необходимо авторизироватьcя.
2)Тыкнуть в поле для ввода пароля.
3)Зайти в PASMAN.
4)Выбрать нужный пароль.
5)Нажать на кнопку "Вставить автоматически".
6)Пароль вставится в нужное поле, и в буфере обмена будет находится ваш логин.
7)Тыкните в поле для ввода логина и нажмите "Ctrl+V"
